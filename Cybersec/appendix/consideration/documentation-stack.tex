\subsection{Documentation Stack}

While I considered developing a full featured project management and CI/CD pipeline 
along with virtualized environments and fit for purpose tools, it seemed overly time
consuming.  This approach of having everything self contained in a single docker 
container and using minimal tools and manual processes is easier to setup, shows my
profeciency in basic principles, and is more cost effective.

In the future I will adopt more specific and featurefull technologies, but for now
this will suffice.

Also, I really enjoy coding and since \LaTeXe{} is essentially documentation via code
it makes it fun and enjoyable!
